About FitTrack

FitTrack is a simple fitness tracking website built as a university project for the Internet Programming course at Altinbas University.


The goal of this website is to help users log their daily workouts, view their exercise history, and explore a library of exercises with useful tips.


The website was built using HTML, CSS, JavaScript, and Supabase as the database.
